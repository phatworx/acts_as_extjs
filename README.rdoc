== ActsAsExtjs

ActiveRecord Store - Helper for Extjs

== Dependecy

* Rails 3

== Installtion

Insert into Gemfile

  gem 'acts_as_extjs'

== Integration

class Model < ActiveRecord::Base
  acts_as_extjs
end
